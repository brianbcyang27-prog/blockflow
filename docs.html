BlockFlow

BlockFlow 用 3 個區塊設計你的每一天：專注 · 個人 · 恢復

🧠

核心概念：3 區塊生活

BlockFlow 幫你把一天分成三個有意義的區塊。每個區塊有不同的目的，你不必同時做所有事。

🔵 專注區塊 Focus

這是深度工作的時間。唸書、寫程式、開會——任何需要專心的事。設定時長，開啟 Pomodoro，開始產出。

🟢 個人區塊 Personal

這是屬於你的時間。彈吉他、看書、運動、學新東西——不需要有生產力，只需要對你有意義。

🟡 恢復區塊 Recovery

這是刻意休息的時間。滑手機、追劇、放空——因為是規劃好的，所以沒有罪惡感。

不是管理時間，而是管理生活模式。 三個區塊讓你輪流專注於每個面向，而不是同時兼顧全部。

🔥

問題

每天滑手機好幾個小時，專注力越來越差，到了晚上卻覺得什麼都沒做。不是你不夠努力——是你的時間缺少一個簡單的架構。

沒有結構的一天，很容易被通知、社群媒體和慣性帶著走。你需要的是設計，不是意志力。

✨

功能

📋 三個區塊，自訂時長

把一天切分成 Focus、Personal、Recovery。每個區塊 15–240 分鐘，自由調整。完成就打勾，進度條給你動力。

🍅 Pomodoro 專注計時器

25 分鐘倒數，幫助你進入心流。開始／暫停／重置，自動累計總專注時間。看到數字累積本身就是激勵。

🤖 AI 智慧排程

NVIDIA LLM 自動分析事件重要性（高／中／低），推薦最佳區塊安排。AI 判斷，使用者也能手動調整——兩者並行。

📊 一鍵分心記錄

忍不住滑手機？按一下「我分心了」。每天看到具體次數，覺察本身就是改變的開始。

😴 睡眠追蹤

記錄睡覺和起床時間，自動計算時長。支援跨夜。一週後你會發現睡眠和專注力的直接關聯。

🔄 每日重置 + 7 天回顧

午夜自動重置，每天都是全新的一天。保留一週歷史，讓你看到模式、發現趨勢、持續進步。

📅

行事曆 × AI 決策

BlockFlow 的手動行事曆搭配 NVIDIA LLM，實現基於重要性的智慧時間分配。

📝 手動輸入事件

原本想整合 Google Calendar API，但啟用需要綁定信用卡。因此改為手動輸入模式——使用者自己新增事件，零成本、零權限問題。

🧠 AI 判斷重要性（自動）

NVIDIA Llama 3.1 模型自動分析每個事件的標題與描述，給出高／中／低三個重要性等級，並建議最適合的區塊。

✋ 使用者手動調整（手動）

AI 的判斷不是最終決定。使用者可以隨時手動覆寫重要性等級與區塊分配——AI 輔助，但控制權永遠在用戶手上。

⏱️ 時間分配建議

基於事件的優先級與期限，AI 推薦每個區塊的最佳時間長度，幫助用戶把精力花在真正重要的事情上。

使用成效

📚 考前衝刺

連續一週用 Focus Block + Pomodoro 唸書。分心次數明顯減少。考試前更有信心。

😌 不再耍廢的週末

週六安排：早上散步（Personal）、下午看書（Focus）、晚上電影（Recovery）。整天充實而不疲憊。

📉 打破熬夜循環

連續幾天睡眠不足的數據擺在眼前。開始固定就寢時間。一週後 Focus 完成率明顯提升。

👀 注意力真相

第一天的分心次數讓人大開眼界。把降低次數當成一場遊戲，一週後明顯減少。

🔄 建立日常儀式

每天早上規劃區塊、晚上打勾確認成為習慣。開始期待每週日的回顧——看到自己一週的節奏。

一整天的 BlockFlow

看看一個學生如何用 BlockFlow 度過完整的一天。

🌅 8:00 AM

設定今天的三個區塊

三個區塊等待填滿。設定：Focus 9–12（唸書）、Personal 14–15（吉他）、Recovery 20–22（追劇）。一天的藍圖完成。

☀️ 9:00 AM

Focus 區塊啟動，Pomodoro 開始

按下 Start Focus，25 分鐘倒數。唸到一半想滑 IG — 按了「分心」。4 個 Pomodoro 後：分心 3 次。明天目標：2 次。

🌤️ 2:00 PM

Personal 區塊：充電時間

自動切換到 Personal Block。彈吉他，沒有壓力。這是設計好的成長時間。

🌙 8:00 PM

Recovery 區塊：安心休息

恢復區塊開始。安心滑手機、追劇。睡前記錄睡眠。

📊 隔天早上

回顧：3/3 區塊完成 ✅

起床記錄睡了 7.5 小時。昨天三區塊全完成。一週後發現：睡眠 <6h 的日子 Focus 完成率低。調整作息，數據說話。

AI 整合

BlockFlow 整合了 NVIDIA API，利用 LLM（大語言模型）來增強生產力體驗。

🧠 智慧事件分析

AI 自動分析使用者輸入的事件，判斷重要性等級（高／中／低），並建議最適合的區塊（Focus、Personal、Recovery）。

⏱️ 排程建議

基於事件的重要性與期限，AI 推薦最有效的區塊安排，幫助使用者把精力花在最重要的事情上。

📝 生產力助手

內建 AI 聊天助手，隨時回答時間管理相關問題、提供客製化建議，幫助使用者建立更好的生活節奏。

🔗 NVIDIA API 整合

使用 NVIDIA Llama 3.1 模型，透過共用代理路徑與 Firebase rewrite 實現瀏覽器端 AI 推論，無需手動串接多個不同 API。

免費的 NVIDIA API Key 即可啟用，無需綁定信用卡。
